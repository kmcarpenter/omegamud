Combat
Commands Use of Command
kill Kill is used to start combat, normally. To do so, you type in kill <creatures_name>.
flee Flee is for the weak or the strategically involved. This command will normally allow you to escape from combat unless you are affected by an ability or room affect that prevents it. When using flee, this will remove you from combat into a randomly selected adjacent room.
cast Before you can cast a spell, you need to practice it (see help Training and Practicing). When casting a spell, the syntax is cast <spell_name> <target>, mind you the <target> is optional. A special note about casting spells is that if the spell name is longer than one word, you need to use single quotes around the spell name (ie cast 'wrath of god' Satan).

There are some spells in the game that are used in combat that do damage or that have affects that either affect yourself or your target creature. To find out about which spells do so, while playing, type help <spell_name>. Also, there are spells that will give you added affects that are not used in combat. The spell help files in the game will tell you whether or not this is possible.

Spells are another alternative to using the kill command to start combat.
wimpy wimpy <number> will allow you to set your 'wimpy' setting to a value of hitpoints that you feel, if you get below, that you should flee. Upon reaching this number set, your character will automatically attempt to flee until your character either perishes or you succeed in fleeing. If you use the wimpy command without a number afterwards, the game will set your 'wimpy' option to 20% of your maximum hitpoints.
murder This command is the only command besides a spell that will allow you to initiate combat with another player in the game. This is also providing that both characters have the PKiller flag. murder can also be used on creatures but it is an annoyance watching creatures constantly shout that they are being attacked, so, please use the kill command when attacking a creature.
zap If you are currently holding a wand in the wear loaction of HOLD and you have practiced the skill WANDS, you can use this command to use the affects that are built into a wand to hit a target. To target someone with this, type zap <target>.
brandish This works exactly the same way as wand, except for a staff.
fire Same as a wand except for a holstered item.
push / drag This command is used to move a player or a creature. To push someone, type push <target> <direction>. You can only push someone it they are not sleeping or resting. When they are sleeping or resting you would use drag <target> <direction>.
There are also skills in the game that have their own syntax of being used. To use a skill, you must use the commands as such, <skill_name> <target>. Again, like spells, there are some skills that are used in combat as well as out and the help files inside the game when you look up the skill will show you which ones are used when. Also, when involved in combat, the <target> is optional.
