Misc. Commands
help
HELP without an argument afterwards will show you a one page list of some of the things that have help files. If you type help with a word/spell/skill afterwards, it will show you the help file on that particular argument, that is providing there is a help file available.

repair
In a smithy, you can repair your equipment for a price (yes, equipment can break on here). The easiest way to repair is by typing REPAIR ALL.

examine
This will let you look into a bag or at a sign or a book...etc. To examine something, just use the command with what you would like to examine as the argument.

get
This command is used exactly how it sounds.

put
PUT <ITEM> <SOMEWHERE>

drop
DROP <SOMETHING>

give
GIVE <SOMETHING> <SOMEONE>

sacrifice
Sacrifice a corpse or any object to the gods for a gold piece or something, not to mention to get it out of the way so that no one else can just pick it up.

wear
This is the command to wear an item.

recite
This is how to use a scroll or parchment.

quaff
This is the command you use to drink a vial or potion...

time
This shows you the 'game time' as well as the last time the MUD was started as well as the current local time of where the server is located.

weather
This is the current forecast for the weather in the game.

where
This shows you where all the other players in the same area as you are located. If you use a name after the command, you can also find a single player or a creature that is in the same area as you.

password
You can use this to change your characters password for when you log on.

title
This allows you to change the text after your name when you check the <wholist>. The way to use this is, TITLE <your text here>.

lock
Lock the door that is in the direction that you use as an argument, this is providing you have the key for it.

unlock
Used the same way as lock but the opposite affect.

open
Open the door that is in direction that you use as an argument.

close
Opposite of open.

wizlist
Show you a list of some of the immortals on Stormgate.

spells
Shows you the cost to use any of the spells that you could currently use.

list
In a shop, this will show you what is for sale.

buy
BUY <ITEM>, need I say more?

sell
SELL <ITEM YOU HAVE>

value
When in a shop, if you use value on an object you have, it will allow the shopkeeper to give you an amount that they will buy it from you for.

inventory
Checks what you have with you.

equipment
Checks what you are wearing.

speak
Allows you to change the current language you are speaking to another by using the new language as an argument.

eat
EAT <FOOD TYPE OBJECT>, you know, just in case you get hungery.

drink
DRINK <AN OBJECT YOU CAN DRINK FROM>, to quench that thirst.

fill
FILL <AN OBJECT YOU CAN DRINK FROM>. After all, these things aren't bottomless...

save
After level 2, you can save your character. The mud will do this on it's own at certain intervals but if you don't trust it, save on your own.

quit
Go ahead...leave the game...you'll be back...

pagelength
Using a number after the command will allow a certain amount of lines to scroll on your screen at once.

learn
This is how you learn new languages. It's done exactly like you would TRAIN or PRACTICE (see Training and Practicing).
