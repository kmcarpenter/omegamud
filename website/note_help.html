Notes
Notes are used as a form of communication with another player that is not currently online or if you wish to make some sort of an announcement.
Commands
Use of the Command
note list This will list all of the most recent notes that you can view.
note read <#> If this command is used without the number of the note inserted at the #, you will read the notes available to you in sequential order. If you place a note number in at the #, you will read that note specifically.
note subject When deciding to write your own note, you must add a subject, and this is the command to do it.
note to This is where you put who the note is to. If you would like everyone to see it, put all. If you put Immortals, only Immortals will be able to read it. Besides those 2 entries, you can also place the name of a character so that they will get the note (You may put more than one name in as such note to Kadence Tyrion Deconce Ahsile -> those 4 people will all be able to read the note).
note + This is used to start a new line in the note (ie note + I am writing this note to...). To end the line hit <ENTER>.
note show Shows your note in progress.
note clear Erases the current note you are working on.
note post or note send Sends/posts your note so that the recipients may read it. Post is not automatic, it must be sent or posted manually.
note remove If you are the sender of the note that you are using this command on, it will remove the note from the postings. This means that people that haven't read it yet will no longer have the chance to read it.
Okay, there is a different may of writing the context of your note rather than using the note + command. Most people find it a pain to read a note that goes all the way across the screen and doesn't automatically word wrap. The replacement that people use for note + is note write. The commands for note write are:

.c This allows you to clear the slate and start again if you wish.
.f This command formats the text to the standard that has been set for length across the screen as well as colors.
.d <line_number> Using this command will allow you to delete a whole line out of your description. This line is decided by when looking at it, what actual, physical line is currently is on.

*** When deciding on which line you are trying to delete, remember that the first line is not line number 1, it is actually line number 0 ***
.s Using this command will show you what your description currently looks like in full.
@ This will allow you to exit the description editor.
.r <word_or_sentence_to_replace> <word_or_sentence_to_be_replaced_with> The way this command works is if you wish to replace a word, use the command as such:
.r demno demon
This will replace the first instance of the word/typo demno with demon.

To do a replace for a sentence:
.r 'eyes lok lke wateer' 'eyes look like water'
This will replace what is in the first set of single quotes (EXACTLY) with what is in the second set of single quotes.
To enter your text for your description, just type in what you wish to have and then to make a new line, hit <ENTER>. It's as simple as that... :) (don't forget to format it after you finish).

Also, if you are using Zmud or any other application that defaults keys to separate functions, before you decide to use the editor, you will need to make sure that none of those keys are changed so that they do not conflict with the editor.
